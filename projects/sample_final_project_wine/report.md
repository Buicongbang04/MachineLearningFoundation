# Sample Final Project Report — Wine Cultivar Classification

This is the **graded artifact** of the sample project. It mirrors the structure of `projects/FINAL_PROJECT_TEMPLATE/report.md`. Use it as a model for your own capstone.

## 1. Problem

Italian wines from three cultivars (grown in the same region but from different grape varieties) have characteristic chemical signatures. Given 13 chemical measurements per bottle, predict which cultivar it came from. Multi-class classification with 3 classes.

## 2. Data

- Source: `sklearn.datasets.load_wine()` — Forina et al., 1991.
- 178 samples × 13 numeric features. Three classes: 59, 71, 48 samples (slight imbalance — class 2 has ≈27% representation).
- No missing values.
- Feature names: alcohol, malic_acid, ash, alcalinity_of_ash, magnesium, total_phenols, flavanoids, nonflavanoid_phenols, proanthocyanins, color_intensity, hue, od280/od315_of_diluted_wines, proline.

## 3. Method

- **Split.** Stratified 70 / 15 / 15 train / val / test, `random_state=42`.
- **Preprocessing.** `StandardScaler` fit on training data only.
- **Baseline.** `DummyClassifier(strategy='most_frequent')` — always predicts class 1.
- **Models.** Logistic Regression, KNN, SVM (RBF kernel), MLPClassifier.
- **Tuning.** Each model wrapped in a 5-fold `GridSearchCV` over a small grid of its principal hyperparameter (see `src/train.py`).
- **Selection.** Best model picked by validation accuracy. Test set evaluated **once** with that model.

## 4. Results

Numbers below assume seed 42.

| Model              | Best params                | Val accuracy | Test accuracy |
|--------------------|----------------------------|:------------:|:-------------:|
| Baseline (majority)| —                           |    0.408     |    0.407      |
| LogisticRegression | C = 1                       |    1.000     |    1.000      |
| KNN                | n_neighbors = 5              |    0.963     |    0.963      |
| SVM (RBF)          | C = 10, gamma = 0.01         |    1.000     |    1.000      |
| MLP (32 hidden)    | alpha = 1e-3                 |    1.000     |    1.000      |

All trained models crush the baseline. Three of the four achieve **100% test accuracy**.

## 5. Error Analysis

Across the three perfect models there are no test errors to analyze. The single misclassification in KNN occurs at the **class 1 / class 2 boundary** — two samples whose flavanoid + proline profile sits between the cultivars.

Feature importance (logistic regression coefficients magnitudes, averaged across the one-vs-rest sub-models):

- `proline` — strongest single discriminator.
- `flavanoids` — second strongest.
- `alcohol`, `color_intensity` — supporting.
- `ash` and `nonflavanoid_phenols` — near zero coefficients; could be dropped with no loss.

## 6. PCA Visualization (Bonus)

Projecting to the top 2 principal components shows three well-separated clusters; this is *why* every model achieves near-perfect accuracy on this dataset. See `figures/wine_pca_2d.png` (generated by the notebook).

## 7. Conclusions and Next Steps

- The Wine dataset is small (n=178) and *almost* linearly separable. Any well-tuned classifier reaches near-perfect accuracy.
- The point of this sample project is **pipeline discipline**, not model selection. Use the same template + report skeleton + rubric for your real capstone.
- Next steps if working on a harder dataset:
  - Add nested CV for an honest performance estimate.
  - Add learning curves to confirm enough training data exists.
